主要用于解决大数据情况下的渲染速度问题。


## API

| 参数      | 说明               | 类型             | 默认值  |
| --------- | ------------------ | ---------------- | ------- |
| listData      | 渲染的子节点，数组形式，包裹一个个组件对象           | `Components[]`         | -       |
| height      | 可视区高度     | `number` | 400       |
| size      | 子节点高度预估值,尽量合理     | `number`         | 30       |
| bufferScale | 缓冲区比例     | `number`         | 1       |
