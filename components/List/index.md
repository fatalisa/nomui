通用列表。

## 何时使用

- 最基础的列表展示，可承载文字、列表、图片、段落，常用于后台数据展示页面。

## API

| 参数      | 说明               | 类型             | 默认值  |
| --------- | ------------------ | ---------------- | ------- |
| gutter      | 栅格间隔      | `'sm'\|'md'\|'lg'`      | -      |
| line      | 分割线、网格线、交叉线      | `'split'\|'grid'\|'cross'`      | -      |
