进行标记和分类的小标签。

## 何时使用

- 用于标记事物的属性和维度。
- 进行分类。

## API

| 参数      | 说明               | 类型             | 默认值  |
| --------- | ------------------ | ---------------- | ------- |
| text      | 文本值      | `'string'`      | -      |
| icon      | 设置图标      | `'string'`      | -      |
| color      | 标签色      | `'string'`      | -      |
| number      | 设置数字      | `'number'`      | -      |
| size      | 设置尺寸      | `'xs'\|'sm'\|'lg'\|'xl'`      | -      |
| type      | 设置标签类型      | `'round'\|'pill'`      | -      |
| removable      | 可删除标签      | `'(key) => {console.log(key)}'`      | -      |
