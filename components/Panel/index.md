归纳整理信息，分类显示。

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| header | 面板头部，可以配置标题以及工具栏 | `object` | - |
| body | 面板内容 | `object` | `{children: '面板内容'}` |
| fit | 让 面板内容 拉伸充满父容器的高度，同时内容超过高度时出现滚动条 | `boolean` | - |
| uistyle | 面板样式 | `'plain'\|'bordered'\|'card'\|'splitline'\|'outline'\|'default'` | default |
