## 何时使用

- 用于纵向布局

## API

### Rows

| 参数      | 说明                             | 类型             | 默认值  |
| --------- | -------------------------------- | ---------------- | ------- |
| items     | 子级对象                         | `array `         | []      |
| gutter    | 栅格间距                         | `sm \| md \| lg` | `md`    |
| showEmpty | 数据为空的时候是否显示空信息提示 | `boolean`        | `false` |
