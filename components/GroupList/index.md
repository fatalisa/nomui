## API

| 参数            | 说明             | 类型     | 默认值 |
| --------------- | ---------------- | -------- | ------ |
| label           | 标题字段         | `string` | -      |
| addDefaultValue | 配置默认值       | `{}`     | -      |
| fields          | 子字段数组       | `[]`     | -      |
| onValueChange   | 数据发生改变回调 | `()=>{}` | -      |

> .tips 可参考字段组
