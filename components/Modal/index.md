## 何时使用

- Modal 模态框

## API

### Modal

| 参数                | 说明                       | 类型             | 默认值  |
| ------------------- | -------------------------- | ---------------- | ------- |
| content             | 模态框内容                 | `Object\|string` | -       |
| closeOnClickOutside | 是否点击外部区域关闭模态框 | `boolean`        | `false` |
