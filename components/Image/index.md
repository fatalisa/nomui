图片

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| src | 图片的资源地址，可使用多张图片的数组或单张图片的 string（如果前面的图片加载失败会依次加载备用图片） | `[] \| string ` | `null` |
| alt | 图片 alt 信息 | `string` | - |
| width | 自定义图片的宽度 | `number ` | - |
| height | 自定义图片的高度 | `number` | - |
| iconWidth | 自定义占位图的宽度 | `number ` | 200 |
| iconHeight | 自定义占位图的高度 | `number` | 100 |
