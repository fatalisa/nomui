返回页面顶部的操作按钮。


## API

| 参数      | 说明               | 类型             | 默认值  |
| --------- | ------------------ | ---------------- | ------- |
| duration      | 回到顶部所需时间（ms）           | `number`         | 100       |
| target      | 设置需要监听其滚动事件的元素，值为该滚动元素的类名（class）     | `string` | body       |
| visibilityHeight      | 滚动高度达到此参数值才出现 BackTop     | `number`         | 400       |
| right      | 操作按钮右边距离     | `string` | body       |
| bottom      | 操作按钮底边距离 B     | `number`         | 400       |
| onClick | 点击按钮的回调函数     | `()=>{}`         | -       |
