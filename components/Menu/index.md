为页面和功能提供导航的菜单列表，常用于网站顶部和左侧。

## API

| 参数      | 说明               | 类型             | 默认值  |
| --------- | ------------------ | ---------------- | ------- |
| direction      | 菜单类型，可选值为 horizontal（水平） 和 vertical（垂直）           | `String`         | vertical       |
| text      | 菜单标题     | `String`         | -       |
| id      | 菜单的唯一标识，必填     | `String`         | -       |
| url | 跳转的链接     | `String`         | -       |
| uistyle      | 鼠标悬浮在菜单上的效果，可选值为 pill       | `String`         | -       |
| byClick      | 激活菜单带选中状态 | `Boolean`         | false       |
