省略溢出的文字

## API

| 参数      | 说明                      | 类型      | 默认值 |
| --------- | ------------------------- | --------- | ------ |
| text      | 内容                      | `string`  | -      |
| showTitle | 是否用 title 显示内容全文 | `boolean` | `true` |
| line      | 文本展示的行数            | `number`  | 1      |
